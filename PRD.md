# 몽실 (MongSil) - AI 꿈 해몽 그림 일기 ☁️

## 🎯 프로젝트 개요
**"오늘 꾼 꿈, 그림으로 남겨보세요."**
사용자가 꿈 내용을 이야기하면 AI가 따뜻한 해몽과 함께 꿈 속 장면을 그림으로 그려주는 감성 다이어리 앱입니다.

**버전:** 1.0.0
**플랫폼:** Flutter (Android / iOS)
**타겟 사용자:** 2030 감성 유저, 꿈에 의미를 두는 사람, 기록 덕후

---

## 📱 핵심 기능 (MVP)

### 1. 📝 꿈 기록 (Dream Log)
- 텍스트 입력 또는 음성 녹음(STT)으로 꿈 내용 기록
- 날짜, 꿈의 기분(이모지) 선택

### 2. 🎨 AI 화가 (Dream Painter)
- 입력된 텍스트를 프롬프트로 변환하여 AI 이미지 생성
- **스타일 선택:** 몽환적인 수채화, 지브리 애니메이션풍, 펜화, 추상화
- 이미지 재생성 기능 (마음에 들 때까지)

### 3. 🔮 AI 해몽 (Dream Analysis)
- 꿈의 심리적 상징과 의미 분석
- 오늘의 운세와 연결된 긍정적인 메시지 제공
- 해몽 결과 텍스트 제공

### 4. 📅 꿈 캘린더 (Dream Calendar)
- 월별 캘린더 뷰에서 그림 썸네일 확인
- 날짜 클릭 시 상세 일기(그림+해몽) 조회

## 💰 수익화 모델 (Business Model)

### Freemium (부분 유료화)
- **Free:** 하루 1회 생성, 광고 노출, 기본 스타일(수채화)만 제공
- **Pro (월 4,900원):** 무제한 생성, 모든 스타일 잠금 해제, 광고 제거, 고화질 다운로드

---

## 🛠️ 기술 스택 (Tech Stack)

- **Framework:** Flutter
- **State Management:** Provider 또는 Riverpod
- **Backend:** Firebase (Auth, Firestore, Storage)
- **AI API:**
  - Image: DALL-E 3 (OpenAI)
  - Text: GPT-4o (OpenAI)
